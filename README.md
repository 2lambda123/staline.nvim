A simple statusline for neovim written in lua.

### Installation
* Vim-plug:
    ```vim
    Plug 'tamton-aquib/staline.nvim'
    ```
* Packer
    ```lua
    use { 'tamton-aquib/staline.nvim' }
    ```

* Paq
    ```lua
    paq 'tamton-aquib/staline.nvim'
    ```

* Dein
    ```vim
    call dein#add('tamton-aquib/staline.nvim')
    ```

### Features
* Lightweigth (2.3 kb)
* Fast
* Unicode mode info.
* 100 LOC.
